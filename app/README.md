# URL shortener API

## Overview

* Node.js Express based web app
* REST API for all operations
* Configured to log to console, but can be changed to any output destination with Bunyan logger

## Development

1. `npm install`
2. `nodemon` or `npm start`
3. Go to `http://localhost:8080/`

## Configuration

Use environment variables (see `nodemon.json` for an example):

* `PORT`: web server port
* `API_KEY` API secret key
